# Expense Meter server

* All api's are accessed using POST
* All requests should contain name and password in body 

# API

## user

** /users **

* /login

## category

** /category **

* / - list categories
* /add - add category
* /edit - edit category
* /delete - delete category


## transactions

** /transactions **

* / - list transactions
* /add - add transactions
* /edit - edit transactions
* /delete - delete transactions
